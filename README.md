# 🎨 PaletteX

A modern 2-color palette generator built with HTML, CSS and JavaScript.

Generate perfectly matching color pairs for websites, apps, branding and UI designs.

---

## ✨ Features

- Generate Matching Color Pairs
- One Click Generation
- Copy HEX Codes
- Responsive Design
- Modern UI
- Fast & Lightweight

---

### 🎨 Generator Page

- 2 Matching Colors
- Generate Button
- Copy Color Support
